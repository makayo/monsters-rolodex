# Monsters Rolodex

A fun and interactive monster encyclopedia web application built with React. Search through a collection of monsters and click on any one to view detailed information including their race, height, weight, diet, home planet, and special abilities.

## Features

- Monster Search - Real-time search functionality to find monsters by name
- Monster Cards - Interactive card grid with hover animations
- Detailed Monster Pages - Click any monster to view:
  - Monster avatar with glowing effects
  - Race and species information
  - Height and weight stats
  - Diet preferences
  - Home planet origins
  - Personality traits
  - Primary and secondary special abilities
  - Danger level rating (1-5)
  - Intelligence rating
  - Age in alien cycles
  - Contact information
- Responsive Design - Works seamlessly on desktop, tablet, and mobile devices
- Beautiful UI - Cyberpunk-themed design with smooth animations and glowing effects

## Tech Stack

- React - UI library
- CSS3 - Advanced styling with gradients, animations, and effects
- JavaScript (ES6+) - Modern JavaScript features
- Create React App - Project bootstrapping

## Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/makayo/monsters-rolodex.git
cd monsters-rolodex
```

2. Install dependencies:

```bash
npm install --legacy-peer-deps
```

3. Start the development server:

```bash
export NODE_OPTIONS=--openssl-legacy-provider && npm start
```

4. Open your browser and navigate to http://localhost:3000/monsters-rolodex

## How to Use

1. Search - Use the search box to find monsters by name
2. Browse - Scroll through the interactive monster card grid
3. Click - Click on any monster card to open the detailed view
4. Explore - View all the monster's stats, abilities, and information
5. Back - Click "Back to Rolodex" to return to the main list
